
## Webpack starter template for local development

Webpack starter template for local development with SCSS, ES6, Pug and Images + Live reload

## How to get started?

1. Clone the repo!
```shell
git clone https://github.com/owlypixel/WebpackStarter_SCSS_ES6_Pug_Images.git
```

2. Navigate into the repo install dependencies and run the development server:
```shell
npm install
npm start
```
